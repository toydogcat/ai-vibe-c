# Python Example

這個資料夾包含兩個 Python 範例程式：`exp1.py` 與 `exp2.py`。

## 內容說明

- `exp1.py`：Python 基礎練習，適合了解語法、流程與基本運算。
- `exp2.py`：進階練習或延伸範例，可用來學習更多 Python 程式設計概念。

## 執行方式

建議使用 Python 3 執行：

```bash
python3 exp1.py
python3 exp2.py
```

若檔案已有執行權限，也可直接執行：

```bash
./exp1.py
./exp2.py
```

## 套件安裝

若程式使用 PIL，請先安裝 Pillow：

```bash
pip install pillow
```

若使用 Python 3 的 pip，請改成：

```bash
python3 -m pip install pillow
```

## 小提醒

- 若遇到權限問題，請先執行 `chmod +x exp1.py exp2.py`。
- 建議使用 `python3 --version` 確認 Python 版本為 3.x。
