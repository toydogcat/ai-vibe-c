
# AI Vibe C step03-frontend-vite - 開始指南

歡迎使用 AI Vibe C！這是一個使用 AI Studio 或其他代碼生成工具來完成前端專案的專案。

## 快速開始

### 環境準備 安裝 Node.js
為了順利執行這個專案，請確保您的開發環境已正確設置。以下是詳細步驟：

- **推薦版本**：Node.js 18 或更高版本（支援最新的 JavaScript 功能和 Vite）。
- **下載方式**：
  - 前往 [Node.js 官方網站](https://nodejs.org/) 下載並安裝 LTS 版本。
  - 或者使用套件管理員安裝（例如，在 macOS 上使用 Homebrew：`brew install node`；在 Ubuntu 上：`sudo apt update && sudo apt install nodejs npm`）。
- **驗證安裝**：安裝後，打開終端機執行以下命令確認版本：
  ```bash
  node --version
  npm --version
  ```
  - 如果版本正確，您應該看到類似 `v18.x.x` 的輸出。
